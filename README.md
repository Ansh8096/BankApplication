# 🏦 BankApplication

🚧 **Status:** In Progress

A backend banking application built using **Spring Boot** with a focus
on clean architecture, security, and real-world banking workflows.

------------------------------------------------------------------------

## 🛠 Tech Stack

-   Java
-   Spring Boot
-   Spring Security (JWT)
-   MySQL / MongoDB (planned)
-   Redis (planned)
-   Apache Kafka (planned)

------------------------------------------------------------------------

## ✨ Current Features

-   Layered architecture (Controller, Service, Repository)
-   Spring Security configuration with JWT foundation
-   User & Role management structure
-   Global exception handling
-   Clean package organization

------------------------------------------------------------------------

## 🚀 Planned Features

-   User & Account management
-   KYC verification flow
-   Transaction handling (credit/debit)
-   Redis caching
-   Kafka-based event processing

------------------------------------------------------------------------

## ▶️ Running the Project

``` bash
mvn clean install
mvn spring-boot:run
```

> ⚠️ `application.yml / application.properties` are intentionally
> ignored to keep credentials secure.

------------------------------------------------------------------------

## 🎯 Goal

To build a **secure, scalable banking backend** following industry best
practices and suitable for backend interviews.
